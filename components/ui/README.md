# UI components

Reusable building blocks. **Do not edit** unless changing design system behavior.

| Component | Use |
|-----------|-----|
| `Section` | Page block wrapper (spacing, grain, divider) |
| `SectionHeader` | Eyebrow + title + description (animated) |
| `ButtonLink` | Primary / secondary / glass CTA links |
| `AnchorLink` | Nav links with optional underline |
| `Reveal` | Scroll-triggered fade-up wrapper |
| `Stars` | Testimonial star row |
| `AnimatedOrb` | Hero background glow |
| `SectionFallback` | Lazy-load placeholder |

Import from `@/components/ui` or individual files.
